# 🛒 SuperKart Sales Forecasting & Model Deployment

## 📌 Project Overview
This project was completed as part of the **Model Deployment** course.

SuperKart is a retail chain operating supermarkets and food marts across multiple cities. The objective of this project is to forecast outlet sales revenue for the upcoming quarter using machine learning techniques and deploy the model for scalable business usage.

---

## 🎯 Objectives
- Analyze historical sales data
- Forecast future outlet sales revenue
- Build and optimize predictive models
- Deploy the model using Flask and Docker
- Enable scalable prediction through APIs
